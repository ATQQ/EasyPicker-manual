## 项目简介
**[轻取](http://sugarat.top/EasyPicker/home)**,为方便在线文件收取而生。

## 项目背景
校园学习或者工作场景中会出现以下几个场景:
1. 碰到上机课的时候,会遇到收取电子版的实验报告。
2. 上级部门下发通知,需要相关人员填写各种电子表格或者电子文档。
3. 每当有比赛时需要收取报名表
4. 有志愿者活动时需要收取报名表
5. 类似场景还有很多很多,这里就不赘述。。。

通常的方式是,通过QQ/微信/邮箱等收取,弊端显而易见,当人数过多时,就会产生太多的消息弹窗,很容易漏收,不仅收集麻烦且不方便整理统计。还占用电脑/手机内存。为了解决这个问题,此项目应运而生。
